# SOCKET-ADAPTERS
Various adapters for ICs

23xxxAdapter: Eprom 27128/27256/27512 (28pin) to a 2332 or 2364 socket (24pin)

27xxx_to_27c080: Eprom 27c010 to 27c080 (32pin) to a 27256 or 27512 socket (28pin)

6502-65816: CPU WDC65C816 (16bit) to a 6502 (VIC-20, PET)

74LS12: Recreated the 7412 (discontinued) with a 7411 and 7406.  Use LS, HCT, AHCT or LV-A.

DIP48-XC9572XL: Xilinx CPLD to fit 48pin for C128 MMU, PLA etc.

LM323K: Triple TO-220 to fit a TO-3 (VIC-20 regulator replacement)

dipXXsoic: Fit soic-XX to dip-XX
dipXXtssop: fit tssop-XX to dip-XX
dip20soic573-373: Fit an SOIC-20 74573 to a DIP-20 74373 socket.
dip20tssop573-373: Fit a TSSOP-20 74573 to a DIP-20 74373 socket.

pi1541translator: Fit a TXS0104 4-bit level translator to a Raspberry Pi 1541 PCB as the translator unit
